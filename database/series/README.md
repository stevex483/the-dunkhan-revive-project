# Series Database

This directory contains the documentation for every known series produced by **Dunkhan & SuperMonkeyMan (TreizsoftGames)**.

Each series has its own dedicated page containing:

- General information
- Recovery status
- Known episodes
- Missing episodes
- Related playlists
- Known reuploads
- Useful notes

The goal of this directory is to provide a centralized overview of every documented series.
