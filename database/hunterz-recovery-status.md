# HunterZ Recovery Status

**Version:** 1.0

**Last Updated:** 2026-07-07

---

## Purpose

This document tracks the recovery status of every known HunterZ episode published on the TreizsoftGames channel.

It serves as the project's central reference for identifying which episodes have been recovered and which remain lost.

---

## Methodology

The status assigned to each episode reflects the best information currently available.

The absence of a recovered copy does not prove that an episode no longer exists.

Recovery statuses may change as new evidence is discovered.

---

## Legend

| Status | Meaning |
|---------|---------|
| ✅ Recovered | A complete playable copy has been confirmed. |
| 🟡 Partial | Only partial footage or excerpts have been recovered. |
| 🔴 Lost | No playable copy is currently known. |
| ⚪ Unknown | Episode existence or status has not yet been confirmed. |
